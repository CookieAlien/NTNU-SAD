### 學習時程規劃
1. 學習區塊鍊及以太坊的基礎知識
2. 上CryptoZombies完成Solidity部分的練習
3. 到Scrima學習React並完成至少1~2項練習的專案\(可能也需要練習Javascript\)
4. 建置完本地的開發環境，在自己的電腦上製作React的網頁
5. 在React網頁基礎上融入Ethers.js,Solidity,Truffle等智慧型合約及虛擬貨幣
